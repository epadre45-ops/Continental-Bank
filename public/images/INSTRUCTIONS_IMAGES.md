## 📁 Emplacement
Placez toutes les images dans : `public/images/`

## 🎨 Recommandations

### Qualité
- **Format**: JPG pour les photos, PNG pour les logos
- **Compression**: Optimisée pour web (qualité 80-90%)
- **Résolution**: 72 DPI pour le web
- **Taille max**: 500KB par image

### Style
- **Cohérence**: Même style photographique sur toutes les images
- **Lumière**: Claire et professionnelle
- **Composition**: Équilibrée avec espace pour le texte
- **Couleurs**: Harmonieuses avec la charte corporate

### Alternatives
Si vous n'avez pas ces images spécifiques, vous pouvez utiliser :

1. **Images de banque** : Bâtiments modernes, intérieurs de banque
2. **Images business** : Personnes en réunion, bureaux modernes
3. **Images technologie** : Appareils mobiles, interfaces digitales
4. **Images corporate** : Architecture moderne, espaces professionnels

## 🔧 Installation

1. **Téléchargez** ou **créez** les images selon les spécifications
2. **Renommez** exactement comme indiqué
3. **Placez** dans le dossier `public/images/`
4. **Vérifiez** que les chemins sont corrects

## ⚡ Fallback Automatique

Le site inclut un système de fallback :
- Si une image ne se charge pas → placeholder automatique
- Si l'image est absente → gradient de remplacement
- Garantie d'affichage optimal même sans images
