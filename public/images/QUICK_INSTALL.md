## 📥 Méthodes d'Installation

### 🖱️ **Méthode 1: Navigateur (Recommandé)**
1. **Copiez** chaque URL ci-dessus
2. **Collez** dans votre navigateur
3. **Clic droit** → "Enregistrer l'image sous..."
4. **Nommez** exactement comme indiqué
5. **Placez** dans `/public/images/`

### 💻 **Méthode 2: Terminal (Automatique)**
```bash
# Téléchargez toutes les images d'un coup
```
