## 🔄 Remplacement Facile

Quand vous avez les vraies images :

1. **Téléchargez** les images réelles
2. **Renommez** selon les noms requis
3. **Placez** dans `/public/images/`
4. **Remplacez** automatiquement les placeholders

Les placeholders seront remplacés automatiquement sans changer le code !
